-----------------------------------------------------
                     Dev Setup
-----------------------------------------------------

1. Fork the repo or download .zip
2. Extract the.zip to a folder
3. Open a terminal in the folder
4. Enter npm install
5. Enter npm start
6. Open a second terminal
7. cd client
8. Enter npm install
9. npm start
10. Happy hacking!
